##Task Manager - MERN Stack Application

A full-stack Task Management application built using the MERN stack.  
Users can create, view, update, and delete tasks with task status management.

##Backend API:
https://task-manager-1-7tzh.onrender.com

##Frontend API:
https://task-manager-nine-rose-74.vercel.app


##Features

- Create new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Change task status
- MongoDB database integration
- REST API backend
- Responsive UI
- Environment-based configuration
- Deployed frontend and backend separately

## Frontend

- React.js
- Vite
- Tailwind CSS
- Axios
- React Hot Toast

## Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- CORS

## Deployment

- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
